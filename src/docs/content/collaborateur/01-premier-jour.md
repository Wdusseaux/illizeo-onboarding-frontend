# Mon premier jour sur Illizeo

Bienvenue dans votre nouvelle entreprise et bienvenue sur Illizeo. Cet espace personnel rassemble tout ce dont vous avez besoin pour votre arrivée : votre parcours d'intégration, vos documents, les contacts utiles. Voici comment démarrer.

## Première connexion

Vous avez reçu un email d'invitation Illizeo à votre adresse personnelle. Cliquez sur le bouton **« Activer mon compte »**.

À la première connexion :

1. Choisissez votre **mot de passe** (minimum 12 caractères)
2. Activez la **double authentification (2FA)** avec votre téléphone
3. Sauvegardez les **codes de récupération** affichés
4. Renseignez vos **préférences** (langue, fuseau horaire, photo)

Votre compte est prêt en moins de 5 minutes.

## La page d'accueil

Une fois connecté, vous arrivez sur votre **tableau de bord personnel**. Vous y voyez :

- Votre **prochain rendez-vous** dans le parcours
- Les **actions à faire** cette semaine
- Vos **derniers documents** ajoutés
- Les **messages** reçus
- Votre **buddy** assigné (si applicable)

## Compléter votre profil

Avant le jour J, vous avez quelques actions de préboarding à compléter. Allez dans **Mon parcours** pour voir la liste.

Typiquement :

- Upload de votre **pièce d'identité** (l'OCR Illizeo extrait automatiquement les données)
- Saisie de votre **RIB / IBAN** pour la paie
- Renseignement de votre **contact d'urgence**
- Choix de la **taille de t-shirt** pour le welcome pack
- Préférences **alimentaires** (allergies, régime)
- Signature du **contrat de travail** électronique

Plus vous complétez tôt, plus l'arrivée est fluide.

> 💡 Astuce : utilisez l'app mobile Illizeo pour photographier votre pièce d'identité directement avec votre téléphone. C'est plus rapide qu'un scan.

## Découvrir l'équipe

Dans **Mon parcours**, vous trouvez aussi :

- Une **vidéo de bienvenue** du CEO (durée 2 à 5 min)
- La **présentation de votre équipe** (photos, prénoms, rôles)
- Un mot de votre **manager direct**
- Un mot de votre **buddy** (votre référent informel)

Prenez le temps de tout regarder avant le jour J. Vous arriverez en connaissant déjà quelques visages.

## Le jour J

Le jour J, plusieurs actions vous attendent :

- Accueil par votre manager
- Tour des locaux
- Petit déjeuner ou café avec l'équipe
- Remise du matériel (laptop, badge, etc.)
- Premier 1:1 manager
- Présentation de votre buddy
- Déjeuner d'équipe

Tout est tracé dans Illizeo. Vous cochez les actions au fur et à mesure.

## Les notifications

Illizeo vous notifie pour :

- Les actions à faire (rappel J-1)
- Les nouveaux documents
- Les messages de votre manager / buddy / RH
- Les rendez-vous (1:1, formations, NPS)
- Les anniversaires (1 semaine, 1 mois, 1 an)

Vous configurez les notifications via **Mon profil → Notifications**.

> ⚠️ Important : ne désactivez pas toutes les notifications. Au moins gardez les notifications du parcours actif. Sinon vous risquez d'oublier des actions critiques (signature, déclarations).

## Application mobile

Illizeo est aussi disponible en application mobile (iOS et Android) :

- Notifications push
- Accès aux documents
- Signature électronique mobile
- Mood checkins en 30 secondes
- Chat avec le buddy / manager

Téléchargez via **App Store** ou **Play Store** en cherchant « Illizeo ».

## Aide et support

En cas de question :

- **Chatbot Illizeo** (bouton flottant bas à droite) : disponible 24/7
- **Mon RH** : voir contact dans **Mon profil → Contacts utiles**
- **Mon buddy** : référent informel pour les questions du quotidien
- **Mon manager** : pour les questions métier
