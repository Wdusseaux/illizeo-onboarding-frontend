# Mes documents et signatures

Tous vos documents professionnels sont centralisés dans votre espace personnel Illizeo : contrat, fiches de paie, attestations, certificats. Plus besoin de fouiller dans les emails ou de demander à votre RH. C'est votre coffre-fort personnel.

## Accéder à mes documents

Allez dans **Mon espace → Mes documents**.

Vous voyez vos documents organisés par catégorie :

- **Contrat** — contrat initial, avenants, lettres de mission
- **Rémunération** — fiches de paie, primes, attestations
- **Identité** — votre pièce d'identité uploadée
- **Formation** — diplômes, certificats internes
- **Onboarding** — documents signés à votre arrivée
- **Administratif** — RIB, contact d'urgence, mutuelle
- **Médical** — visite médicale (accès limité)

## Consulter un document

Cliquez sur un document pour le **prévisualiser** dans le navigateur.

Vous pouvez :

- **Télécharger** le PDF
- **Imprimer** (icône en haut)
- **Demander une copie certifiée** (avec tampon employeur, sur demande)
- Voir la **date d'émission** et le **statut** (signé, en attente, archivé)

## Signer un document

Quand un document à signer arrive :

- Notification **in-app** + email
- Cliquez sur le **lien de signature**
- Vous êtes redirigé vers **DocuSign** ou **UgoSign**
- Lisez le document en entier
- Signez avec votre code SMS ou votre identité numérique
- Le PDF signé revient automatiquement dans votre espace Illizeo

> 💡 Astuce : avant de signer, lisez attentivement. Une signature électronique a la **même valeur juridique** qu'une signature manuscrite. Demandez-vous toujours « est-ce que je signerais ce papier au stylo ? ».

## Les types de signatures

Selon le document, le niveau de signature varie :

- **Signature simple** : un clic, signature de base (ex. règlement intérieur)
- **Signature avancée (AES)** : avec code SMS, valable juridiquement (contrat de travail)
- **Signature qualifiée (QES)** : avec identité numérique vérifiée (rare, pour les actes très importants)

## Refuser de signer

Vous avez le droit de **refuser une signature** si :

- Vous ne comprenez pas le document
- Vous êtes en désaccord avec une clause
- Vous voulez consulter un proche / juriste avant

Cliquez sur **Refuser** et indiquez le motif. Votre RH est notifié et reprend contact avec vous.

> ⚠️ Important : refuser de signer un contrat de travail à l'embauche peut bloquer votre arrivée. Discutez d'abord avec votre RH avant de cliquer sur « Refuser » officiellement.

## Uploader un document

Pour les documents à fournir (RIB, diplôme, justificatif de domicile) :

1. Cliquez sur **Mon parcours → Action « Upload [type] »**
2. Choisissez le fichier (PDF, JPG, PNG)
3. Si applicable, l'IA OCR extrait les données
4. Validez

Les documents uploadés sont stockés dans la GED, accessibles à votre RH.

## Demander un document à mon RH

Pour les documents que vous n'avez pas encore (attestation employeur, copie d'un diplôme interne) :

1. **Mes documents → Demander un document**
2. Choisissez le type
3. Indiquez le motif (banque, location, administration)
4. Validez

Votre RH reçoit la demande et y répond généralement sous 48h ouvrées.

## Versions et historique

Si un document a été modifié (avenant, nouvelle fiche de paie corrigée), vous voyez :

- La **version actuelle** en tête de liste
- L'**historique** complet (cliquez sur « Historique »)
- La **comparaison** entre deux versions

## Confidentialité

Vos documents sont **strictement personnels** :

- Le RH y a accès (pour les besoins administratifs)
- Le manager **n'y a pas accès** par défaut (sauf documents partagés volontairement)
- Vos collègues n'y ont **aucun accès**

L'audit log trace toutes les consultations. Voir [Audit log](?article=admin-audit-log).

## Récupération à mon départ

Au moment de quitter l'entreprise, vous avez **30 jours** pour télécharger l'intégralité de vos documents :

1. **Mes documents → Export complet**
2. Un ZIP est généré contenant tous vos PDF
3. Téléchargez et conservez (cloud personnel, NAS, clé USB)

Au-delà de 30 jours, vos documents restent chez Illizeo (pour les obligations légales de conservation 10 ans), mais vous ne pouvez plus y accéder en libre-service. Contactez votre ancien RH.

> 💡 Astuce : faites cet export **avant** votre dernier jour, pas après. Vous aurez moins de temps et moins de motivation après le départ.
