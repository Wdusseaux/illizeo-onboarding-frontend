# Les 8 IA Illizeo

Illizeo embarque 8 intelligences artificielles propulsées par **Claude (Anthropic)**, hébergé sur des centres de données européens. Elles automatisent les tâches répétitives, détectent les signaux faibles et augmentent la productivité de votre équipe RH de 30 à 50 %.

## Pourquoi Claude

Nous avons choisi **Claude d'Anthropic** plutôt qu'OpenAI ou Mistral pour 4 raisons :

- **Hébergement européen** disponible (conformité RGPD plus simple)
- **Excellence en français** — modèle multilingue avec qualité élevée
- **Constitutional AI** — alignement éthique fort, refuse les usages détournés
- **Vision native** — pas de modèle séparé pour l'OCR, tout dans Claude

Aucune de vos données n'est utilisée pour entraîner les modèles Anthropic.

## Plans IA et quotas

Les fonctions IA consomment des **crédits IA** mensuels selon votre plan :

- **Starter** : 29 CHF/mois — Chatbot + OCR (50 OCR/mois)
- **Business** : 79 CHF/mois — toutes les 8 IA (quotas généreux)
- **Enterprise** : 199 CHF/mois — quotas illimités

Le module **AiUsageGuard** monitore vos quotas en temps réel et envoie une alerte à 80 % de consommation.

## Auto-recharge et crédits supplémentaires

Si vous dépassez votre quota mensuel :

- **Auto-recharge** : achat automatique de packs (10 CHF / 100 unités) si activé
- **Achat ponctuel** : pack de crédits achetable manuellement via **Facturation → Crédits IA**
- **Refus** : la fonction est temporairement désactivée jusqu'au mois suivant

> 💡 Astuce : activez l'auto-recharge pour les périodes de pic (recrutements massifs en septembre par exemple). Vous évitez les blocages au mauvais moment.

## Confidentialité et sécurité

Toutes les fonctions IA respectent :

- Chiffrement TLS 1.3 en transit
- Pas de stockage côté Anthropic
- Logs anonymisés pour le debug
- Possibilité de désactiver feature par feature
- Pas d'utilisation des données pour entraînement

Voir [RGPD](?article=admin-rgpd) pour le détail.

## Désactiver l'IA

Si votre politique interne interdit l'IA, vous pouvez :

- Désactiver feature par feature via **Paramètres → IA**
- Désactiver toutes les fonctions IA (impact sur l'expérience produit)
- Demander un déploiement on-premise (Enterprise uniquement)

> ⚠️ Important : désactiver l'IA n'a pas d'impact sur votre abonnement de base. Vous pouvez désactiver l'IA et garder le module RH/onboarding complet.
