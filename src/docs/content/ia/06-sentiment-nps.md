# Analyse sentiment NPS

Recueillir des verbatims dans les NPS et exit interviews est facile. Les **analyser systématiquement** est une autre histoire. L'IA Illizeo analyse les milliers de verbatims que vous collectez chaque année et fait remonter automatiquement les thèmes, les évolutions et les recommandations.

## Le problème : les verbatims dorment

Sans analyse IA, les verbatims restent :

- **Bloqués** dans des fichiers Excel ou des PDF
- **Lus** par 1 ou 2 personnes seulement
- **Oubliés** au-delà de 3 mois
- **Inactionnable** parce que non synthétisés

L'IA résout ce problème : elle lit, classe, synthétise et alerte.

## Ce que l'IA analyse

L'IA Illizeo passe en revue les verbatims de :

- **NPS d'arrivée** (J+30, J+90)
- **Mood checkins** hebdomadaires / bi-mensuels
- **Exit interviews** (parties écrites)
- **Feedbacks ad hoc** (canal de feedback hub)
- **Conversations avec le chatbot** (anonymisées)

## Les outputs de l'analyse

Pour chaque période (semaine, mois, trimestre), l'IA produit :

- **Thèmes récurrents** classés par fréquence
- **Sentiment global** (positif / négatif / mixte)
- **Mots-clés émergents** vs périodes précédentes
- **Citations significatives** (verbatims marquants, anonymisés)
- **Recommandations** actionnables

## Exemple de rapport mensuel

> **Top thèmes positifs (mars 2026)** :
> 1. Qualité du buddy (mentionné 47 fois) — sentiment : très positif
> 2. Welcome pack (32 mentions) — positif
> 3. Outils tech (28 mentions) — positif
>
> **Top thèmes négatifs** :
> 1. Charge de travail première semaine (23 mentions) — négatif
> 2. Manque de clarté objectifs (15 mentions) — négatif
>
> **Émergent ce mois** : commentaires sur la cantine (12 mentions, neuf cas)
>
> **Recommandation IA** : alléger la charge de la première semaine, surtout pour l'équipe Tech (15 sur 23 mentions viennent de Tech).

## Accéder aux analyses

Allez dans **Espace admin → Insights → Analyse sentiment**.

Vous pouvez filtrer par :

- Période (mois, trimestre, année, custom)
- Equipe / département
- Manager
- Type de feedback (NPS, mood, exit)
- Pays / site

## La détection de signaux faibles

Au-delà des thèmes globaux, l'IA détecte des **signaux faibles** :

- Un mot-clé qui apparaît 3 fois cette semaine alors qu'il n'apparaissait jamais
- Une équipe dont le sentiment chute de 1 point en 2 semaines
- Un manager dont les feedbacks deviennent significativement plus négatifs

Ces signaux génèrent des **alertes** au RH responsable.

> 💡 Astuce : ne réagissez pas à un signal faible isolé. Un seul verbatim négatif n'est pas un pattern. Attendez 3 à 5 occurrences sur 2 semaines pour considérer comme un vrai signal.

## L'anonymisation

Les verbatims publiés dans les rapports sont **toujours anonymisés** :

- Pas de nom, pas d'initiales
- Pas d'élément contextuel (équipe précise, projet précis) qui pourrait identifier
- Si une équipe a moins de 5 personnes, ses verbatims sont **regroupés** avec une équipe voisine

C'est la condition pour que les collaborateurs continuent à donner des feedbacks sincères.

> ⚠️ Important : ne demandez jamais à votre équipe RH de « retrouver qui a dit ça ». Cela tue le programme. Si un sujet émerge, traitez-le **collectivement** sans chercher l'auteur.

## Comparaison entre périodes

L'IA peut comparer deux périodes (ex. Q1 2026 vs Q1 2025) :

- Évolution du NPS global
- Nouveaux thèmes émergents
- Thèmes qui ont disparu (succès d'une initiative)
- Évolution par équipe

Cela permet de **mesurer l'impact** des actions RH dans le temps.

## Action sur les insights

Chaque insight peut être transformé en **action** dans Illizeo :

- Créer un **workflow** pour suivre une dimension
- Lancer un **mini-sondage** ciblé
- Organiser un **focus group** sur le thème
- Modifier un **template de parcours** pour adresser le problème
