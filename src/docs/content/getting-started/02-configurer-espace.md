# Configurer mon espace

À votre première connexion, Illizeo lance automatiquement un assistant de configuration en 6 étapes. Cet assistant prend environ 10 minutes et vous permet d'avoir un espace fonctionnel pour démarrer votre premier onboarding dès aujourd'hui.

## L'assistant de configuration

L'assistant (wizard) s'ouvre dès votre première connexion en tant qu'administrateur. Vous pouvez aussi le relancer à tout moment via **Paramètres → Configuration initiale**.

Les 6 étapes :

1. **Informations société** — nom légal, forme juridique (Sàrl, SA, SARL...), n° IDE/TVA, adresse, pays.
2. **Identité visuelle** — logo, couleur principale, favicon.
3. **Équipes et entités** — création de vos départements et entités juridiques si vous êtes un groupe.
4. **Premiers utilisateurs** — invitation des admins RH et managers.
5. **Intégrations clés** — connexion Microsoft 365 ou Google Workspace pour le SSO.
6. **Premier parcours** — copie d'un template d'onboarding pour démarrer.

## Informations société

Renseignez avec soin votre **numéro IDE** suisse (format CHE-123.456.789) ou votre numéro de TVA européen. Ces informations apparaissent sur :

- Les factures d'abonnement Illizeo
- Les contrats de travail générés
- Les en-têtes de documents officiels

Si vous opérez dans plusieurs pays, ajoutez chaque entité juridique séparément. Cela permet de générer des contrats locaux avec les bonnes mentions légales.

> 💡 Astuce : si vous n'avez pas encore tous les détails (n° AVS du gérant, etc.), vous pouvez compléter plus tard via **Paramètres → Société**. Les champs obligatoires sont marqués d'une astérisque rouge.

## Fuseau horaire et langues

Illizeo est multilingue (FR, EN, DE, IT, ES) et multi-devise (CHF, EUR, USD, GBP, CAD, AUD, JPY). Définissez :

- La **langue par défaut** de votre espace (celle de l'interface admin)
- Les **langues activées** pour les collaborateurs (chacun peut basculer)
- Le **fuseau horaire principal** (Europe/Zurich par défaut)
- La **devise principale** pour les rapports financiers

## Vérification des données

Avant de cliquer sur « Terminer », l'assistant affiche un récapitulatif. Vérifiez en particulier :

- L'adresse légale (utilisée sur les contrats)
- Le n° IDE (impossible à changer sans validation manuelle)
- Le logo (format PNG transparent recommandé, 512×512 px minimum)

> ⚠️ Important : une fois l'assistant terminé, votre espace est en **mode démo** pendant 14 jours. Vous pouvez tout tester sans saisir de carte bancaire. Pour passer en production, activez votre abonnement via **Paramètres → Facturation**.

## Et après ?

Une fois l'assistant terminé, vous arrivez sur le **Dashboard admin**. Trois actions à faire en priorité :

1. [Personnaliser le branding](?article=personnaliser-branding) en détail
2. [Inviter votre équipe RH](?article=inviter-equipe-rh)
3. [Créer votre premier parcours d'onboarding](?article=onboarding-creer-parcours)

Si vous bloquez à une étape, contactez le support via le bouton « ? » en bas à droite. Notre équipe répond en moins de 4 heures ouvrées.
